# 📊 Power BI Ecommerce Sales Dashboard

## 📌 Project Overview
This project is an interactive Ecommerce Sales Dashboard built using Microsoft Power BI. It helps analyze sales performance, profit, customer trends, and regional performance through interactive visualizations.

## 🚀 Features
- Sales Performance Analysis
- Profit Analysis
- Category-wise Sales
- State-wise Sales
- Customer Insights
- Interactive Slicers & Filters
- KPI Cards

## 🛠️ Tools & Technologies
- Microsoft Power BI
- DAX
- Power Query
- Microsoft Excel

## 📂 Dataset
- Orders.csv
- Details.csv

## 📈 Key KPIs
- Total Sales
- Total Profit
- Total Quantity
- Average Order Value
